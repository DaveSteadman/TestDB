# TestDB

A repository with a polished web UI interface for managing ID/Text pairs.

## Client

The `/Client` directory contains a modern React web application for displaying and managing ID/Text pairs.

### Features
- High-end, polished UI design
- Responsive layout
- Animated backgrounds and smooth transitions
- Built with React, Vite, and Tailwind CSS

### Quick Start

```bash
cd Client
npm install
npm run dev
```

For more details, see the [Client README](Client/README.md).
